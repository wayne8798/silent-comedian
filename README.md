CS 467 - the Silence Project
============================

I used ffmpeg to extract audio track from the mp4 file. A sample command is:
`ffmpeg -i my_video.mp4 output_audio.wav`

Libraries required by backend:
[OpenCV](http://opencv.org/), 
[NumPy](http://www.numpy.org/), 
[SciPy](http://www.scipy.org/)
